This is a project carried out as part of the Web Developer training program at OpenClassrooms.

This project is an integration of dynamic elements, using JavaScript to reduce the HTML code, via requests to the back end of the website.

My role was to optimize the website, reduce the HTML code, by using JavaScript and fetch requests the call the API.

Specifications

Technical constraints:
Reduce the HTML code to work mainly with JavaScript.
Respect the filters on the images, to ensure they are listed correctly when selected.
Add a login page, to ensure that only the owner of the website can modify their listed projects.
Follow the design model to be sure that it's respecting what's the owner wants.
Add the login option for the owner, once connected, there as to be options where they can add, modify or delete their projects.
