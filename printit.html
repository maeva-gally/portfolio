PrintIt

This is a project carried out as part of the Web Developer training program at OpenClassrooms.

This project is an implementation of an interactive banner carousel using HTML, CSS, and JavaScript. It allows you to create an image carousel that can be used to display promotional images, advertising banners or any other visual content that you want to highlight on your website.

My role was to create the website by integrating the carousel in JavaScript and making the site dynamic and more attractive.

Specifications

Technical constraints:
Analize the HTML and CSS elements already in the code to reuse it on the website.
Add the arrows to navigate on the carousel.
Add an event listener on each arrows.
Add bullet points on the lower part of the slider, one for each images.
The points have to signal which images you're looking at.
The bullets ppints have to change depending on the dirrecction of the arrows, following left and right.
The images on the carousel will have to slide infinitely, the last one leading to the fist one if you click on the right arrow.
