Kasa

This is a project carried out as part of the Web Developer training program at OpenClassrooms.

This project is the replacement of the legacy code of the website that was in ASP.NET, using React and React router.

My role was to create routes using React and React router, to upgrade the code of the website, making it more secure and modern.

Specifications

Technical constraints:
Use React router to ensure that the pages are correctly linked to the main one.
Add a 404 error page, for when the route doesn't exist.
Add a carousel for the "housing" section.
Use Sass for the styles of the website.
