ArgentBank

This is a project carried out as part of the Web Developer training program at OpenClassrooms.

This project is an implementation of a login solution for users, and display their bank account transactions, as well as the beginning of API routes, to complete the transactions part.

My role was to implement the website, create a user interface, and integrate the possibility to view transactions and change nicknames, without mixing up users informations, with the help of Mongo Database.

Specifications

Technical constraints:
Implement the website as wished by the contractor.
Implement a login interface for users to consult their bank accounts.
Display transactions of users, with no possibility for the others to see without being logged in or if it's not their own account.
The users should be able to modify their username, but not the first or last names.
An error message, for when users do not enter correct informations when they try to login.
Use Redux to manage the states values, simplifying errors detections and better debugging.
Optimizing the code as well as the images of the website, to ensure quicker loading times, and respect environmental friendly practices.
Create API routes with Swagger, not functional for now, for the users to see more details about their transactions.
